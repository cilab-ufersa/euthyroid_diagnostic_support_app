# Euthyroid Diagnostic Support Application 


## Prerequisites

What things you need to have to be able to run:

  * Python 3.6 +
  * Pip 3+
  * VirtualEnvWrapper is recommended but not mandatory

## Requirements 

```bash
$ pip install -r requirements.txt
```

## Run 

```bash
$ streamlit run  main.py
```
